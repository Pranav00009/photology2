📋 OPTION 2: GITHUB + NETLIFY (BEST)
-------------------------------------
Step 1: Create GitHub Repo
- Go to: https://github.com/new
- Name: photlogy-website
- Click "Create repository"

Step 2: Upload Code
Open terminal in photology folder:

cd E:\photology\photology
git init
git add .
git commit -m "Initial commit"
git remote add origin https://github.com/YOUR_USERNAME/photlogy-website.git
git push -u origin main

Step 3: Connect to Netlify
- Go to: https://app.netlify.com/
- Click "Add new site"
- Choose "Import from Git"
- Select GitHub
- Choose your repo
- Click "Deploy site"

Done! Auto-deploy enabled ✅
