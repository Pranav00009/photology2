📋 3-STEP QUICK START
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

STEP 1: ADD IMAGES
──────────────────
Place these images in assets/images/ folder:

  ✓ hero-silhouette.jpg (1920x1080px)
  ✓ about-image.jpg (1200x800px)
  ✓ portfolio-1.jpg to portfolio-6.jpg (800x1000px)

📸 Need images? See IMAGE-GUIDE.md for free sources


STEP 2: UPDATE LINKS
─────────────────────
Open index.html and update:

  Line ~150: Instagram reel URLs
    data-reel="https://www.instagram.com/reel/YOUR_REEL_ID/"

  Line ~280: WhatsApp number
    href="https://wa.me/YOUR_NUMBER"

  Line ~285: Instagram handle
    href="https://instagram.com/YOUR_HANDLE"


STEP 3: OPEN & TEST
───────────────────
  1. Double-click index.html
  2. Website opens in browser
  3. Test all sections and links
  4. Check mobile view (resize browser)


🚀 DEPLOY TO WEB
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

EASIEST: Netlify (Free)
  1. Go to netlify.com
  2. Drag & drop the 'photology' folder
  3. Your site is live!
  4. Get free URL: yoursite.netlify.app

OTHER OPTIONS:
