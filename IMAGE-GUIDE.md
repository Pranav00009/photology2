## 🛠️ Image Preparation & Optimization

### Step 1: Download Images
- Download highest resolution available
- Save in a temporary folder first

### Step 2: Resize Images
Use free tools:
- **[Squoosh](https://squoosh.app)** (web-based, recommended)
- **[TinyPNG](https://tinypng.com)** (compression)
- **Photoshop** (if you have it)
- **GIMP** (free Photoshop alternative)

**Resize to exact dimensions:**
- Hero: 1920x1080px
- About: 1200x800px  
- Portfolio: 800x1000px

### Step 3: Optimize File Size
Target file sizes:
- Hero image: 150-250KB
- About image: 100-150KB
- Portfolio images: 80-120KB each

**Optimization Tools:**
- [TinyPNG](https://tinypng.com) - Compress without quality loss
- [Squoosh](https://squoosh.app) - Advanced compression options
- [ImageOptim](https://imageoptim.com) - Mac app for batch optimization

### Step 4: Apply Cinematic Color Grading (Optional)

**Quick Adjustments:**
- Increase warmth (add orange/yellow tones)
- Slightly decrease saturation for muted look
- Increase contrast
- Add slight vignette (darker edges)
- Reduce highlights, lift shadows

**Free Editing Tools:**
- **Photopea** (web-based Photoshop alternative)
- **GIMP** (desktop, free)
- **Canva** (web-based, easy to use)
- **Lightroom Mobile** (free app)

**Preset Recommendations:**
- Look for "cinematic" or "moody" presets
- Orange and teal color grading
- Warm sunset tones
- Film-like grain (subtle)

If creating custom images:

**Equipment Needed:**
- DSLR or good smartphone camera
- Natural light or softbox
- Tripod (optional)

**Shooting Tips:**
- Golden hour (sunset/sunrise) for warm tones
- Shoot in RAW for better editing
- Multiple angles and compositions
- Backlit silhouettes for drama
- Use manual focus for sharpness

**Editing:**
- Warm color temperature (3500-4500K)
- Orange/gold color grading
- Increase contrast
- Add subtle vignette
- Sharpen slightly

## 💡 Pro Tips

### Color Consistency
- Edit all images with similar color grading
- Maintain warm tones throughout
- Use same contrast levels
- Keep brightness consistent

### File Naming
- Use lowercase
- No spaces (use hyphens)
- Descriptive names
- Example: `hero-silhouette.jpg`, not `IMG_1234.jpg`

### Backup Originals
- Keep original high-res versions
- Store in separate folder
- Never overwrite originals
- Easy to re-export if needed

### Test on Mobile
- Images should look good on small screens
- Important elements should be visible
- Text should be readable
- Colors should pop

### Legal Considerations
- Only use images you have rights to
- Check license terms
- Give attribution if required
- Get client permission for their work
