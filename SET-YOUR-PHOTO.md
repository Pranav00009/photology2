# 📸 Apni Photo Website Mein Kaise Set Karein

## 🎯 Quick Steps (2 Minutes)

### Step 1: Photo Save Karein
Apni photo (photographer with camera wali) ko save karein apne computer mein.

### Step 2: Rename Karein
Photo ko rename karein:
- **Hero Section ke liye:** `hero-silhouette.jpg`
- **About Section ke liye:** `about-image.jpg`

### Step 3: Copy Karein
Dono photos ko copy karein is folder mein:
```
photology/assets/images/
```

### Step 4: Refresh Karein
Browser mein website ko refresh karein (F5 press karein)

---

## 📁 File Locations

```
photology/
└── assets/
    └── images/
        ├── hero-silhouette.jpg     ← Aapki main photo (hero section)
        └── about-image.jpg         ← Aapki photo (about section)
```

---

## 🎨 Photo Requirements

### Hero Section Photo (hero-silhouette.jpg)
- **Size:** 1920x1080px ya larger
- **Style:** Portrait with camera
- **Background:** Dark ya black background best hai
- **Lighting:** Side lighting ya rim light effect

### About Section Photo (about-image.jpg)
- **Size:** 1200x800px ya larger
- **Style:** Professional portrait
- **Background:** Dark ya studio setup

## 🚀 Alternative Method (If Photos Not Showing)

Agar photos show nahi ho rahi, to:

1. Check karein file names exactly match kar rahe hain:
   - `hero-silhouette.jpg` (not Hero-Silhouette.JPG)
   - `about-image.jpg` (not About-Image.JPG)

2. Check karein photos `assets/images/` folder mein hain

3. Browser cache clear karein:
   - Press `Ctrl + Shift + R` (Windows)
   - Press `Cmd + Shift + R` (Mac)

---

## 💡 Pro Tips

### Best Photo Style:
- Dark background with warm lighting
- Professional pose with camera
- Clear face visibility
- High resolution (at least 1920px wide)

### Editing Tips:
- Add slight warm color grading
- Increase contrast slightly
- Add subtle vignette
- Keep background dark
